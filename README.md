# Curso de Aprendizaje No Supervisado - Maestría en Economía Aplicada (MeCA) 

Este repositorio contiene los talleres de la materia de Aprendizaje No Supervisado (ANS) durante el primer semestre del año 2026. Los proyectos implementan técnicas de reducción de dimensionalidad mediante Análisis de Componentes Principales (PCA) y Descomposición en Valores Singulares (SVD). Implementación de Clusters a partir de K-means, K-medioides y ...
